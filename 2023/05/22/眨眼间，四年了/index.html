眨眼间，四年了

忙里偷闲想尝试建站，毕竟老网民了，没有自己的个人网站总感觉少点什么东西。

无奈一番折腾下来，发现终究败给了冗长的备案流程，以及高昂的云服务器费用。

突然想到了 Github 免费博客，于是决定用它作为网站主页，并先注册了 zfqy.top 域名。

打开尘封的博客一看，好家伙，原来四年前，当自己刚踏入博士生涯的时候，就在琢磨这个事情。

感叹时间实在是如白驹过隙，四年间，自己磨练了耐心，学会了沟通，也收获了一些微不足道的文章、专利。

更重要的，是我在南京认识的核心朋友们，疫情期间他们的陪伴让我难忘，百度地图上遍布的金星就是最好的见证。

倒大可不必为逝去而担忧，毕竟美好的记忆会永远存在。

朝前看，为了未来更多的美好而启航，毕业设计，请加油！
